Flexbox 屬性

display: flex flex: 主要是 main axis 與 cross axis 構成 main axis > 左右向 cross axis > 上下向 那麼有一些設定屬性，如下。 flex-direction 設定主軸 (main axis) 的方向 .container { display: flex; flex-direction: row; // 橫向 flex-direction: row-reverse; // 橫向，反序排列 flex-direction: column; // 主軸變直向 flex-direction: column-reverse; // 主軸變直向, 反序排列 } justify-content 與主軸 (main axis) 的對齊方式 .container { display: flex; justify-content: flex-start; // 靠左 justify-content: flex-end; // 靠右 justify-content: center; // 置中 justify-content: space-between; // 貼齊父容器，每個子元素的間格一致 justify-content: space-around; // 與父容器的間格為，每個元素間格的一半 justify-content: space-evenly; // 每個元素與父容器都是相同的間格 } align-items 交叉軸 (cross axis) 的對齊方式 …
